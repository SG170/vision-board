# My College Coding Projects 🚀

Welcome to my repository! This contains two of my recent software projects built with different tech stacks, ranging from Operating Systems theory to modern Web Development.

## 1. Bank Queue Simulation (OS/CAO Concept) 🏦
A Python desktop application that demonstrates the **First-Come, First-Served (FCFS)** CPU Scheduling algorithm using a Bank Queue analogy.

**Features:**
- Native UI built with `Tkinter`
- Calculations for Waiting Time & Turnaround Time
- Data generation and dynamic tables
- Generates informative bar charts using `matplotlib` 

**To Run:**
```bash
cd bank_queue_simulation
pip install -r requirements.txt
python main.py
```

## 2. Vision Board Setup (Web Development) ✨
A modern, front-end web application intended for creating and customizing vision boards! Currently features a beautiful, glassmorphic login interface built entirely with the newest styling technologies.

**Tech Stack:**
- React (via Vite)
- Tailwind CSS v4 
- Node / npm

**To Run:**
```bash
cd vision_board/frontend
npm install
npm run dev
```
